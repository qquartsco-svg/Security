# BLOCKCHAIN_INFO

이 저장소는 분산 합의형 블록체인을 구현하지 않는다.

여기서 말하는 “블록체인”은 다음 의미에 가깝다.

- 파일 무결성 체크섬
- 변경 기록의 연속성
- 서명 매니페스트를 통한 검증 가능성

구성:

- `SIGNATURE.sha256`
  - 저장소 주요 파일의 SHA-256 체크섬 매니페스트
- `PHAM_BLOCKCHAIN_LOG.md`
  - 개념/버전/문서/레이어 확장 로그

목적:

- 리포지토리 무결성 확인
- 문서와 코드가 어떤 버전 시점에서 정렬되어 있는지 추적
- 기초 보안 레이어 연구 저장소의 신뢰성 강화

주의:

- 이것은 암호화폐형 체인도 아니고
- 노드 합의형 체인도 아니며
- 네트워크 분산 검증 체인도 아니다

한 줄로 말하면:

**이 저장소의 블록체인 표기는 “로컬 저장소 무결성 추적과 버전 연속성 확인”을 뜻한다.**
